# docs/

> L2 | 父级: [../CLAUDE.md](../CLAUDE.md)

## 模块功能综述

- 主要功能：目录「docs/」接收架构讨论与规范沉淀输入，按主题组织文档并输出可检索的语义索引；输入是设计决策、术语定义与演进记录，输出是面向开发者与 Agent 的规则文档；本目录不承载运行时代码。
- 核心文件：`viewer-spec.md`（规范文档）
- 实现原理：以主题文档作为阅读入口，通过同级文档链接形成可追溯的知识网络；失败路径主要是链接失效或术语不一致，需要在评审时修复。
- 相关文件：上游规范 [../CLAUDE.md](../CLAUDE.md)；同级协作文件见“成员清单”；下游依赖包括 `viewer-spec.md`。
- 调用链路：语义链路（非运行时） `viewer-spec.md` -> 输出

## 成员清单

- [`viewer-spec.md`](./viewer-spec.md)：文档文件，记录该模块规范与说明
